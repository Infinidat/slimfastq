slimfastq
=========

A very efficient lossless compression for fastq files. 

Usage
-----

<pre>
% slimfastq *fastq-file* *new-sfq-file*   : compress *fastq-file* to *new-sfq-file* 
% slimfastq *sfq-file*                    : decompress *sfq-file* to stdout
% slimfastq -h                            : get help
</pre>
